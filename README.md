# TOEIC Study 📚

> **목표:** 기록을 통해 **TOEIC 점수와 영어 실력**을 동시에 올리는 레포입니다.

---

## 목표

- **기본 품사(명사, 동사, 형용사, 부사)**를 정확하게 구별할 수 있다.
- **기본 문장 구조 (S + V, S + V + O, S + V + C)** 를 이해하고 직접 만들어본다.
- TOEIC **파트 5(문법)** 기초 문제를 안정적으로 풀 수 있다.
- 자주 나오는 **필수 단어를 체계적으로 정리**하고, 문장 속에서 사용할 수 있다.
- 하루 공부 내용을 **GitHub 커밋으로 기록**하면서, 공부 습관을 만든다.

---

## 공부 방식

1. **매일 최소 1커밋**을 목표로 한다.
   - `vocabulary/`에 단어 정리
   - `grammar/`에 오늘 배운 문법 정리
   - `practice/`에 문제 풀이 + 해설 정리
   - `daily-log/`에 오늘 공부한 내용 요약

2. **단어 공부**
   - 하루 **20~30개**를 목표.
   - 외울 때 **"뜻만" 아니라 예문까지** 같이 적는다.

3. **문법 공부**
   - 한 번에 많은 문법을 보기보다는  
     **하루에 한 개 핵심 문법**만 제대로 정리.
   - 예: `be동사`, `일반동사`, `주어-동사 일치`, `품사 구별`, `전치사 in/on/at` 등.
   - 각 문법마다
     - 개념 설명
     - 패턴 정리
     - 예문
     - 자주 하는 실수/주의점
     을 함께 정리한다.

4. **문제 풀이**
   - TOEIC 파트 5 또는 기초 문법 문제를 풀고
   - `틀린 문제 + 왜 틀렸는지 이유`를 반드시 적는다.
   - “정답 암기”가 아니라,  
     **문법/품사를 이해하고 맞출 수 있게** 만드는 것이 목표.
